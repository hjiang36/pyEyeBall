Illuminant Simulation¶

class pyEyeBall.Source.Objects.Illuminant.Illuminant(file_name='D65.mat', wave=None)[source]¶

Illuminant light simulation and computation

This class stores illuminant spectra distribution data and does computations and analysis from there. In current version, there is no spatial or temporal varying features in this class.

name¶

str – name of the illuminant instance

photons¶

numpy.ndarray – quanta distribution in each wavelength samples

__init__(file_name='D65.mat', wave=None)[source]¶

Constructor of Illuminant class This function loads illuminant spectra data from ISETBIO .mat illuminant file. The file should contain two variables: data and wavelength

Parameters:
  • file_name (str) – illuminant file name, e.g. D65.mat. The data file should be seated in Data/Illumination
  • wave (numpy.ndarray) – wavelength samples to be used. If None, or not specified, the default wavelength samples np.arange(400, 710, 10) will be used

Examples

Create different illuminant >>> il = Illuminant(“D65.mat”) >>> il = Illuminant(“D50.mat”)
